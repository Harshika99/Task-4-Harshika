 # Currency Converter

A simple Java-based Currency Converter that converts an amount from one currency to another using predefined exchange rates. The application allows users to select source and target currencies, enter an amount, and view the converted value through a menu-driven console interface.

** Features**
- Multiple currency support
- Predefined exchange rates
- User-friendly console interface
- Accurate currency conversion

** Technologies Used**
- Java
- Scanner Class
- Switch Case
- Arithmetic Operations

 **Learning Outcomes**
- User Input Handling
- Conditional Statements
- Currency Conversion Logic
- Java Fundamentals
